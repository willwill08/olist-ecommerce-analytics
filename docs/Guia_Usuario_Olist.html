Guia do Usuário - Análise E-commerce Olist
🏠 Home Page
Página inicial com apresentação visual do projeto e navegação para as demais seções. Contém o logo e identidade visual da Olist, além de KPIs principais do negócio para uma visão geral rápida dos indicadores mais importantes.
👨‍👩‍👧‍👦 Customers (Clientes)
Análise abrangente do comportamento e perfil dos clientes. Inclui distribuição geográfica com mapa interativo, análise demográfica por estado e cidade, comportamento de compra com frequência e valor, e segmentação de clientes por valor e recência. Permite identificar mercados prioritários, padrões de comportamento e oportunidades de expansão geográfica.
👥 Sellers (Vendedores)
Análise de performance e distribuição dos vendedores da plataforma. Contém rankings e métricas de vendas, distribuição regional dos vendedores, análise de produtos por vendedor, e métricas de qualidade com avaliações dos clientes. Identifica top performers, oportunidades de expansão da base de vendedores e estratégias de desenvolvimento.
🎯 Leads
Acompanhamento do funil de vendas e análise de conversão de leads. Visualiza o processo de qualificação e fechamento, taxa de conversão, tendências temporais de geração e conversão, e performance por origem de captação. Identifica gargalos no funil, otimiza processos de qualificação e analisa ROI por canal de marketing.
Interação
Indica que o usuário pode interagir com o visual realizando filtros, segmentações e inserindo informações. Use os controles de filtro para refinar os dados exibidos e obter insights mais específicos sobre diferentes segmentos do negócio.
Drill Down
Permite selecionar um dado específico para detalhar em nível inferior. Clique em qualquer elemento do gráfico para explorar os dados em maior profundidade, como visualizar detalhes de um estado específico ou período de tempo.
Drill Up
Sobe um nível na hierarquia de dados. Use este recurso para retornar a uma visualização mais ampla após ter feito drill down em dados específicos.
Filtros e Segmentações
Mostra os filtros aplicados ou removidos. Use os painéis de filtro para segmentar os dados por período, região, categoria de produto, status do pedido ou qualquer outra dimensão disponível no modelo.
Modo Foco
Destaca uma área específica dos dados para análise. Use este recurso para focar em um visual específico, ocultando temporariamente outros elementos da página para uma análise mais detalhada.
Informativo
Onde estiver este ícone indica que ao passar o cursor sobre o visual será exibido mais detalhes. Use tooltips para obter informações adicionais sobre métricas, definições e contexto dos dados apresentados.
Mais Opções
Clique para acessar opções adicionais, como:
  • Exportar Dados: Permite a exportação dos dados em formato **XLSX** (máximo 150 mil linhas) ou **CSV** (máximo 30 mil linhas).
  • Mostrar Como Tabela: Exibe os dados do gráfico ou visual em formato de tabela, facilitando a visualização detalhada de todos os valores.
  • Destacar: Use para destacar um conjunto de dados ou valores no gráfico, facilitando a visualização de informações específicas.
  • Classificar em Ordem: Organize os dados em ordem crescente ou decrescente com base nos valores selecionados, como maior para menor ou vice-versa.
Navegação entre Páginas
Use os botões de navegação no canto inferior direito para alternar entre as diferentes páginas do relatório. Cada página oferece uma perspectiva específica dos dados: Home, Customers, Sellers e Leads.
Atualização de Dados
Os dados são atualizados automaticamente conforme configurado. Para forçar uma atualização manual, use o botão de refresh no Power BI Service ou aguarde o próximo ciclo de atualização programado.
Acesso Mobile
O relatório é otimizado para visualização em dispositivos móveis. Use o aplicativo Power BI Mobile para acessar os dados em smartphones e tablets, mantendo a funcionalidade completa de filtros e interações.
